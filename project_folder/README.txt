【共有フォルダの運用ルール】
　0.要約
　　‐共有フォルダの運用ルールは大まかに下記になります。詳しくは下記の各項目をご参照ください。
　　　①ファイル、フォルダの作成、更新、削除は必要に応じて各自おこなってください。
　　　②PRJ全体に影響するもの、知っておく必要があるものはハングアウトなどで情報共有するようにしてください。
　　　③ファイルの格納先、フォルダ構成は月末の週で棚卸をおこないます。

　1.利用目的
　　‐共有フォルダは下記の用途でご利用ください。
　　　1-1. お客様との資料授受
　　　1-2. メンバ間での一時的なファイル共有

　2.ファイル、フォルダの作成
　　‐ファイル、フォルダの作成は自由です。
　　　※個々人が必要と感じたら作成してください。
　　　※ファイルの格納先、フォルダ構成は、下記【共有フォルダの棚卸】にて都度整理していきます。
　　
　3.ファイル、フォルダの更新
　　‐ファイル、フォルダの作成は自由です。
　　　※他メンバに影響があるものの場合、更新前後にハングアウトなどで情報共有をお願いします。
　　　※ファイルの格納先、フォルダ構成は、下記【共有フォルダの棚卸】にて都度整理していきます。

　4.ファイル、フォルダの削除
　　‐不要なファイル、フォルダは最上位層にある99_trashに格納してください。
　　‐共有フォルダの棚卸の時に、相談し削除していきます。

【共有フォルダの棚卸】
　・月末の週でフォルダの棚卸を行います。
　・棚卸時、ファイル、フォルダの削除は、99_trashに格納されたものを都度相談後、削除します。
　・棚卸後、ファイルの格納場所やフォルダ構成が変更されることがありますので、ご了承願います。
